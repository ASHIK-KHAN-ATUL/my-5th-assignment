Important FAQ'S

What is Document Object Model (DOM)?

The DOM is a tree-like structure of an HTML document.

How do you select an element from the DOM?

Use `getElementById`, `querySelector`, or `querySelectorAll`.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation adds listeners to parent elements for better performance.

How do you manipulate an element's attributes and styles using the DOM?

Use `setAttribute`, `getAttribute`, or `style` to manipulate attributes and styles.
